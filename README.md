# Oxygen

This is a prompt GPT like tool which I created in 2019 during my 3rd year academics using C++ language . 
It consist of following features :
1. Text to speech
2. Custom command additions.
3. Search anything on Internet
4. Open any music files
5. Shutdown/restart
6. Show my IP
   
In order to run this program you will need to install codeblocks with necessary dependencies and make sure MinGW is installed too

Scope : n - You can add as many features and functions you want further in to this code .
